Moje umiejętności

Zajmuję się tworzeniem aplikacji na wszystkie urządzenia, od stron internetowych, aplikacji backendowych i webowych po aplikacje mobilnych, kładąc szczególny nacisk na funkcjonalność oraz estetykę wykonania. Dbam zarówno o warstwę interfejsu użytkownika jak i o kod wykonywany po stronie serwera.
Korzystam z narzędzi takich jak:

  • Spring - pozwala mi upraszczać oprogramowanie.
  • Hibernate - dzięki któremu baza danych tworzona jest automatycznie dla aplikacji.
  • GIT - umożliwia mi zapis postępów w pracy i umożliwia szybki powrót w razie zmiany koncepcji.
  • JIRA - pomaga mi zarządzać projektami.
  • JUnits - pozwala na testowanie kodu wtrakcie jego tworzenia.
  • REST API - udostępnia API z którego mogą korzystać inne aplikacje.
  • Maven - udostępnia wiele bibliotek które przyśpieszają pracę.

Odpowiadam również za:

  • Projektowanie, wdrażanie i utrzymywanie aplikacji opartych na języku Java
  • Tworzenie, testowanie, wdrażanie i utrzymywanie oprogramowania aplikacyjnego
  • Rekomendowanie zmian w celu usprawnienia ustalonych procesów aplikacji java
  • Zapewnienie niezakłóconego działania strony WWW czy aplikacji.
  • Troskę o bezpieczeństwo danych.
  • Nadawanie funkcjonalności.
